Historia do HTML 4.0

O HTML 4.0, lançado em 1997, foi um grande avanço na web, trazendo mais flexibilidade, acessibilidade e suporte aprimorado a CSS. Ele introduziu três variações (Strict, Transitional e Frameset), facilitando a transição para padrões mais modernos. Foi a última grande versão antes do XHTML e HTML5, preparando o caminho para uma web mais estruturada e interativa.

Principais pontos do HTML 4.0:
✅ Melhoria na acessibilidade – Introduziu atributos como alt em imagens e title para elementos.
✅ Maior suporte a CSS – Incentivou o uso de folhas de estilo para separar conteúdo e apresentação.
✅ Formulários mais avançados – Novos atributos para melhorar a interação com o usuário.
✅ Depreciação de tags obsoletas – Estilos inline e elementos de formatação visual começaram a ser desencorajados.

Historia do XHTML

O XHTML (Extensible HyperText Markup Language) foi lançado em 2000 pelo W3C como uma evolução do HTML 4.0, trazendo regras mais rígidas baseadas no XML. Ele buscava tornar as páginas mais estruturadas e compatíveis com diferentes dispositivos.

✅ Baseado em XML – Exigia uma sintaxe mais rigorosa, como fechamento correto de tags (<"br"/> ao invés de <"br">).
✅ Código mais organizado e padronizado – Reduziu erros comuns no desenvolvimento web, tornando as páginas mais consistentes.
✅ Maior compatibilidade com diferentes dispositivos – Facilitou a adaptação para celulares, tablets e outros dispositivos.
✅ Fim das tags depreciadas – Elementos antigos do HTML 4.0 foram removidos para incentivar o uso do CSS na formatação.
